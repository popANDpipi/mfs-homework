### 1.什么是媒体查询？我们可以使用媒体查询完成哪些功能？
    媒体类型允许你指定文件将如何在不同媒体呈现。
    该文件可以以不同的方式显示在屏幕上，在纸张上，或听觉浏览器等等。
### 2.响应式断点指的是什么？如何声明响应式断点？
    响应式就是在不同的分辨率下，都要让内容以最佳的展现形式呈现给你用户，提升用户体验。
    如下：
    ```
    // 等于或大于 34*16 = 544px（手机横屏）
    @media (min-width: 34em) { … }
    ```
